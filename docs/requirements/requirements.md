# AutoCM Studio — 要件定義 (Phase 1)

## 背景 / 目的
旧 `auto-cm`（Express + 静的HTML / Miyabiテンプレ）を、Vercel で動く世界最高品質の
Webアプリとしてゼロから再構築する。商品画像1枚から、複数のAIエンジンを選択・比較して
プロ品質のCM動画を生成・ダウンロードできる体験を提供する。

## ペルソナ
1. **個人事業主・EC運営者**: 商品写真しかない。低コストで短尺CMが欲しい。
2. **SNSマーケター**: 縦型(9:16)動画を量産したい。エンジン比較で最適を選びたい。
3. **制作会社のクリエイター**: 高品質(Sora2/Veo3)で、台本を微調整して仕上げたい。

## 機能要件
| ID | 要件 |
|----|------|
| FR-1 | 商品画像のアップロード（ドラッグ&ドロップ / クリック、JPG/PNG） |
| FR-2 | AIによる商品解析（商品名・カテゴリ・特徴・ターゲット・トーン・配色） |
| FR-3 | CM台本の自動生成（フック・ナレーション・動画生成プロンプト） |
| FR-4 | 言語(ja/en/zh)・尺(8/12/15秒)・比率(9:16/16:9/1:1)の指定 |
| FR-5 | 台本（ナレーション・プロンプト）のユーザー編集 |
| FR-6 | 動画生成エンジンの選択（単一/複数同時=比較） |
| FR-7 | おすすめエンジンの自動提示（商品カテゴリ・人物有無に基づく） |
| FR-8 | 非同期での動画生成と進捗表示（ジョブ単位） |
| FR-9 | 生成結果のプレビュー・ダウンロード・再生成 |
| FR-10 | デモモード（APIキー未設定時はサンプル動画で全フロー体験） |

## 対応エンジン
Sora 2 (Replicate) / Veo 3 (Gemini) / Kling (Replicate) / Seedance (ModelArk) / HeyGen
